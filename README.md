## 1. What is the difference between var, let, and const?

Ans: The difference is in their scope and whether their values can be updated or redeclared.


## 2. What is the spread operator (...)?

Ans: It is used to expand the elements of an array or object.


## 3. What is the difference between map(), filter(), and forEach()?

Ans: map and filter create a new array, forEach does not; map transforms items, filter checks conditions.

## 4. What is an arrow function?

Ans: A shorter syntax for functions that does not have its own `this`.

## 5. What are template literals?

Ans: A way to easily include variables inside strings.
